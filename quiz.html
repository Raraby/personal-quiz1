リンクを埋め込む
aを使うことでページ内や別のページのリンクにジャンプすることができます。
リンクがそのまま表示されるのではなく、a要素で挟まれた文字が表示され、クリックすることでページを移動します。
こんなかんじ
使用場所
body内部
属性
href
URLを指定します。
<a href="URL"> ... <a>
target
表示先を指定します。
例えば、新しいタブや新しいウィンドウで開くなどの指定ができます。
<a target="値"> ... <a>
値 : _blank, _parent, _self, _top
_brank : 新しいウィンドウとして、今開いているページの次に表示します。
_self : 今開いているウィンドウで表示します。いつものです。
_parent : 親フレームに表示します。
_top : 最上位のフレームに分割を解除して表示します。
     _parentと_topはむずかしいですx_x
download
リンク先のデータをダウンロードします。
hreflang
リンク先の言語を表します。
ping
リンクにPOSTリクエストを送信します。
情報処理に使用できます。
発展知識
ページ途中の特定の場所を表示する
このページ先頭の目次のように、ページの途中にジャンプする方法です。
  1. htmlファイルでid属性を使います。
    タグの位置はh1要素、div要素、p要素など、どれでもOKです。
    例として、このように書きます。
    htmlファイル
    使用する要素 : div
    使用する属性 : id
    1. <div id="example"> ... </div>
    exampleと書いたところには好きな名前をつけてください。
  2. ジャンプ元のa要素の属性に href="#id名"をいれます。
    例として、このように書きます。
    htmlファイル
    使用する要素 : a
    使用する属性 : href
    1. <a href="#table"> ... </a>
    このページのなかでtableというidは目次につけられているので、次のリンクをクリックすると目次にジャンプすることができます。
    ここをクリックすると目次に飛びます!
また、違うページの時はこのように書きます。
htmlファイル
使用する要素 : a
使用する属性 : href
  1. <a href="飛びたいリンク#example"> ... </a>
exampleのところを何も書かずに#のみにすると、ページのトップにジャンプすることができます。
同じページのときはわざわざページのリンクを貼り付ける手間が省けます！
(違うページのときはページのURLをつけるだけと変わらないので同じページ内の時に効果を発揮します。)
htmlファイル
使用する要素 : a
使用する属性 : href
  1. <a href="#"> ... </a>
ここをクリックするとページトップに飛びます！
